# Task Start Checklist

- [ ] Run `git status --short`.
- [ ] Identify whether the task touches firmware, app, docs, scripts, or tests.
- [ ] Search narrowly with `rg` or GitNexus.
- [ ] Read `task.md` first for firmware/controller upgrade work.
- [ ] Note unrelated dirty files and leave them alone.
- [ ] For complex changes, explain options before editing.
